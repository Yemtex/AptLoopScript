AptLoopScript
=======

Just a simple script, which runs _**`apt`**_ with an for loop twice.  

Script does apt:  

  * _**`update`**_
  * _**`upgrade`**_
  * _**`full-upgrade`**_
  * _**`autoremove`**_

Introduction:  

1.  ```wget https://raw.githubusercontent.com/yyesilkaya/AptLoopScript/master/up.sh```  
2.  ```sh up.sh```
